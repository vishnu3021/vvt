Python Roadmap ‚Äî 10-Week Modules ‚ú®

Structured path from fundamentals to advanced topics. Click any week to expand details.
10-week plan

Week 1 ‚Äî Python Basics

Installation, syntax, variables, operators, basic I/O.

  • Install Python, VS Code, pip, virtualenv
  • Syntax, variables, types (int, float, str, bool)
  • Input/output, string formatting, basic scripts
  • Mini project: Simple calculator & pattern printing

Week 2 ‚Äî Control Flow & Data Structures

Conditionals, loops, lists, tuples, sets, dicts.

  • if/else, nested conditions
  • for & while loops, break/continue
  • List/dict comprehensions
  • Mini project: Student marks analyzer

Week 3 ‚Äî Functions & Modules

Functions, higher-order functions, modules, file I/O.

  • Defining functions, args/kwargs, return values
  • Lambda, map, filter, reduce
  • Creating and importing modules & packages
  • File handling: open/read/write, JSON

Week 4 ‚Äî Object-Oriented Programming

Classes, inheritance, dunder methods and design basics.

  • Classes, instances, constructors
  • Inheritance, polymorphism, encapsulation
  • Magic methods (__str__, __repr__, __eq__)
  • Mini project: Library management system

Week 5 ‚Äî Standard Library & Utilities

OS, datetime, itertools, decorators, generators.

  • OS, shutil, pathlib, datetime
  • Iterators, generators, decorators
  • Working with CSV, JSON, XML
  • Mini project: File organizer CLI

Week 6 ‚Äî Databases & APIs

SQL basics, SQLite/MySQL, requests, building APIs with Flask.

  • SQL CRUD, joins, indexes
  • sqlite3/MySQL connector usage
  • Requests library, consuming APIs
  • Build a basic REST API with Flask

Week 7 ‚Äî Web Development (Flask)

Routing, templates, forms, authentication basics.

  • Flask app structure, routing
  • Jinja templating, static files
  • Form handling, sessions, simple auth
  • Mini project: Login system + dashboard

Week 8 ‚Äî Testing, Debugging & Deployment

pytest, logging, Docker, CI/CD basics.

  • Unit testing with pytest, test structure
  • Logging, pdb, debugging techniques
  • Dockerize an app, GitHub Actions basics
  • Deploy to Render/Railway/Heroku

Week 9 ‚Äî Advanced Python & Performance

Typing, async, concurrency, design patterns, profiling.

  • Type hints & typing module, mypy basics
  • Asyncio: event loop, async/await, tasks
  • Concurrency: threading vs multiprocessing
  • Profiling & optimization (cProfile, timeit), memory tips
  • Common design patterns in Python

Week 10 ‚Äî Capstone & Career

Build a capstone, portfolio, interview prep and open-source.

  • End-to-end capstone: plan ‚Üí build ‚Üí test ‚Üí deploy
  • Documentation, README, hosted demo (Streamlit/Flask)
  • Resume, GitHub portfolio, interview practice
  • Contribute to open-source, keep learning (advanced topics)
Tip: Expand weeks as you learn. Do the mini-projects ‚Äî they matter more than theory alone.

Version Control ‚Äî Git & GitHub (Detailed)

From local commits to advanced collaboration flows and CI integration.

Core Concepts

  • Repository lifecycle: init, clone, commit, branch
  • Branching strategies: Git Flow, trunk-based development
  • Merging vs Rebase and when to use each

Remote Collaboration

  • Remote origins, fetch, push, pull
  • Pull requests, code review, protecting branches
  • Forking workflow, managing contributors

Advanced

  • Stashing, cherry-pick, revert, bisect
  • Git hooks, submodules, large file storage (LFS)
  • CI with GitHub Actions ‚Äî run tests and lint on PR

Hands-on Roadmap

  1. Setup local repo & push to GitHub
  2. Create branches & PRs, do code reviews
  3. Set up CI checks (GitHub Actions) for testing
